1. Introduction aux polynômes

Un polynôme est une expression algébrique constituée d'une somme finie de termes, chacun étant le produit d'une constante (le coefficient) et d'une ou plusieurs variables élevées à une puissance entière non négative.

Définition

Un polynôme à une variable \(x\) de degré \(n\) s'écrit sous la forme :

\(P(x) = a_nx^n + a_{n-1}x^{n-1} + \ldots + a_1x + a_0\)

où \(a_n \neq 0\) et \(a_0, a_1, \ldots, a_n\) sont des nombres réels appelés coefficients du polynôme.

Exemples de polynômes

  • \(P_1(x) = 3x^2 - 5x + 2\) est un polynôme de degré 2
  • \(P_2(x) = x^3 + 4\) est un polynôme de degré 3
  • \(P_3(x) = -2x^4 + 3x^3 - 7x + 1\) est un polynôme de degré 4

2. Opérations sur les polynômes

Addition et soustraction

Pour additionner ou soustraire deux polynômes, on additionne ou soustrait les coefficients des termes de même degré.

Exemple :

\(P_1(x) = 3x^2 - 5x + 2\)

\(P_2(x) = 2x^2 + 3x - 1\)

\(P_1(x) + P_2(x) = (3x^2 - 5x + 2) + (2x^2 + 3x - 1)\)

\(P_1(x) + P_2(x) = 5x^2 - 2x + 1\)

Multiplication

Pour multiplier deux polynômes, on multiplie chaque terme du premier polynôme par chaque terme du second, puis on regroupe les termes semblables.

Exemple :

\(P_1(x) = x + 2\)

\(P_2(x) = x^2 - 3\)

\(P_1(x) \times P_2(x) = (x + 2) \times (x^2 - 3)\)

\(P_1(x) \times P_2(x) = x \times x^2 - x \times 3 + 2 \times x^2 - 2 \times 3\)

\(P_1(x) \times P_2(x) = x^3 - 3x + 2x^2 - 6\)

\(P_1(x) \times P_2(x) = x^3 + 2x^2 - 3x - 6\)

3. Factorisation des polynômes

La factorisation consiste à exprimer un polynôme comme un produit de polynômes de degrés inférieurs.

Méthodes de factorisation

3.1. Mise en facteur commun

Exemple :

\(P(x) = 3x^3 + 6x^2 - 9x\)

\(P(x) = 3x(x^2 + 2x - 3)\)

3.2. Identités remarquables

  • \((a + b)^2 = a^2 + 2ab + b^2\)
  • \((a - b)^2 = a^2 - 2ab + b^2\)
  • \((a + b)(a - b) = a^2 - b^2\)

Exemple :

\(P(x) = x^2 - 4\)

\(P(x) = x^2 - 2^2\)

\(P(x) = (x + 2)(x - 2)\)

3.3. Théorème du reste et factorisation par (x - a)

Si \(a\) est une racine du polynôme \(P(x)\), alors \(P(x)\) est divisible par \((x - a)\).

Exemple :

\(P(x) = x^3 - 4x^2 + 5x - 2\)

On vérifie que \(P(1) = 1 - 4 + 5 - 2 = 0\)

Donc \(x - 1\) est un facteur de \(P(x)\)

\(P(x) = (x - 1)(x^2 - 3x + 2)\)

\(P(x) = (x - 1)(x - 1)(x - 2)\)

\(P(x) = (x - 1)^2(x - 2)\)

4. Division euclidienne

La division euclidienne de deux polynômes \(P(x)\) et \(D(x)\) consiste à trouver deux polynômes \(Q(x)\) (quotient) et \(R(x)\) (reste) tels que :

\(P(x) = D(x) \times Q(x) + R(x)\)

avec \(R(x) = 0\) ou \(\text{deg}(R) < \text{deg}(D)\).

Méthode de division

  1. Ordonner les polynômes par ordre décroissant des puissances.
  2. Diviser le terme de plus haut degré du dividende par le terme de plus haut degré du diviseur.
  3. Multiplier le diviseur par le résultat obtenu et soustraire du dividende.
  4. Répéter jusqu'à ce que le degré du reste soit inférieur au degré du diviseur.

Exemple :

Diviser \(P(x) = 2x^3 - 5x^2 + 3x - 1\) par \(D(x) = x - 2\)

Étape 1: \(2x^3 \div x = 2x^2\)

Étape 2: \(2x^2 \times (x - 2) = 2x^3 - 4x^2\)

Étape 3: \(2x^3 - 5x^2 - (2x^3 - 4x^2) = -x^2 + 3x - 1\)

Étape 4: \(-x^2 \div x = -x\)

Étape 5: \(-x \times (x - 2) = -x^2 + 2x\)

Étape 6: \(-x^2 + 3x - 1 - (-x^2 + 2x) = x - 1\)

Étape 7: \(x \div x = 1\)

Étape 8: \(1 \times (x - 2) = x - 2\)

Étape 9: \(x - 1 - (x - 2) = 1\)

Donc \(P(x) = (x - 2)(2x^2 - x + 1) + 1\)

Quotient: \(Q(x) = 2x^2 - x + 1\)

Reste: \(R(x) = 1\)

5. Exercices interactifs

Exercice 1: Addition de polynômes

Calculez la somme des polynômes suivants :

\(P(x) = 2x^3 - 4x^2 + 3x - 5\) et \(Q(x) = -x^3 + 2x^2 - 3x + 1\)

Solution :

\(P(x) + Q(x) = (2x^3 - 4x^2 + 3x - 5) + (-x^3 + 2x^2 - 3x + 1)\)

\(P(x) + Q(x) = 2x^3 - x^3 - 4x^2 + 2x^2 + 3x - 3x - 5 + 1\)

\(P(x) + Q(x) = x^3 - 2x^2 - 4\)

La réponse correcte est : \(x^3 - 2x^2 - 4\)

Exercice 2: Factorisation

Factorisez le polynôme suivant :

\(P(x) = x^3 - 4x^2 + 4x\)

Solution :

\(P(x) = x^3 - 4x^2 + 4x\)

Factorisation par \(x\) :

\(P(x) = x(x^2 - 4x + 4)\)

Reconnaître l'identité remarquable \((x - 2)^2 = x^2 - 4x + 4\) :

\(P(x) = x(x - 2)^2\)

La réponse correcte est : \(x(x - 2)^2\)

Exercice 3: Division euclidienne

Effectuez la division euclidienne du polynôme \(P(x) = x^3 - 3x^2 + 2x - 5\) par \(D(x) = x - 2\) et donnez le quotient et le reste.

Solution :

Division de \(P(x) = x^3 - 3x^2 + 2x - 5\) par \(D(x) = x - 2\) :

Étape 1: \(x^3 \div x = x^2\)

Étape 2: \(x^2 \times (x - 2) = x^3 - 2x^2\)

Étape 3: \(x^3 - 3x^2 - (x^3 - 2x^2) = -x^2 + 2x - 5\)

Étape 4: \(-x^2 \div x = -x\)

Étape 5: \(-x \times (x - 2) = -x^2 + 2x\)

Étape 6: \(-x^2 + 2x - 5 - (-x^2 + 2x) = -5\)

Donc \(P(x) = (x - 2)(x^2 - x) + (-5)\)

Quotient: \(Q(x) = x^2 - x\)

Reste: \(R = -5\)

La réponse correcte est : Quotient = \(x^2 - x\) et Reste = \(-5\)

Exercice 4: Racines d'un polynôme

Trouvez les racines du polynôme suivant :

\(P(x) = x^3 - 6x^2 + 11x - 6\)

Solution :

Pour trouver les racines, nous allons essayer de factoriser le polynôme.

Essayons de trouver une racine en testant quelques valeurs simples :

\(P(1) = 1 - 6 + 11 - 6 = 0\)

Donc \(x = 1\) est une racine et \((x - 1)\) est un facteur.

Divisons \(P(x)\) par \((x - 1)\) :

\(P(x) = (x - 1)(x^2 - 5x + 6)\)

Factorisons \(x^2 - 5x + 6\) :

\(x^2 - 5x + 6 = (x - 2)(x - 3)\)

Donc \(P(x) = (x - 1)(x - 2)(x - 3)\)

Les racines sont \(x = 1\), \(x = 2\) et \(x = 3\).

La réponse correcte est : 1, 2, 3
